## W3C标准

误区：认为某些元素自带默认样式。

- 元素本身不会带任何默认样式，它的目的只有一个，就是结构
- 所有元素的默认样式都是`浏览器`给的并且解析的
- 浏览器兼容性,指不同浏览器下同一个样式的表现形式不一样
- 元素的默认位置,顶着父级左上角

HTML5：w3c组织在2014年对web开发标准作出的第五次规范修改。

- HTML,XML,XNTML 结构化标准语言
- CSS 变现标准语言
- JS，DOM 行为标准语言

标签语义化：每个标签都有其代表的意义

- h1标签的语义:标题
- div标签的语义:块，盒子，容器
- li标签的语义:列表
